## 🛠 기술 스택
- **언어:** Python 3.x 🐍
- **프레임워크:** python-telegram-bot
- **패키지:**
  - `python-dotenv`
  - `urllib3`
- **운영체제:** Linux 🐧

## 📌 기능
- 🚀 Telegram을 사용하여 원격으로 Linux 셸 명령 실행
- 🔒 사전 정의된 사용자 ID를 통한 보안 접근 제어
- ⚡ 환경 변수를 통한 간편한 설정

## 📥 설치 방법
저장소를 클론하고 필요한 패키지를 설치하세요:
```sh
git clone https://github.com/Blue-B/Botix.git
./Botix/Scripts/activate
pip install -r requirements.txt
```

## ⚙️ 설정
`.env` 파일을 생성하고 아래 환경 변수를 설정하세요:
```sh
Bot_Token="YOUR_TELEGRAM_BOT_TOKEN"  # str
U_Number= Set_ALLOWED_User_Number,1,2,3...  # int
```

## 🚀 사용법
Linux 환경에서 봇 실행:
```sh
python bot.py
```
Telegram에서 아래 명령어를 입력하여 원격으로 셸 명령을 실행하세요:
```sh
/s [your_command]
```
